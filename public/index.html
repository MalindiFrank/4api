4-API Server

This 4-API allows you to:

  • /words - Manage fetching words (id, word, definition, figure)
  • /quotes - Manage fetching quotes (id, author, quote)
  • /colors - Manage fetching color (id, bg, color)

Available Methods:

  • GET - Fetch all items
  • GET - Fetch a random item
  • GET - Fetch an item by ID
  • GET - Fetch a certain number of random items

Endpoints

Available Methods

Routes for each collection (quotes, colors, words):

  • GET /<collection> → all items + count
  • GET /<collection>/:id → one item by ID
  • GET /<collection>/random → one random item
  • GET /<collection>/random/:n → n random items + count

1. Words

Base URL: /words

Description: Fetch words along with their definition and grammatical figure.

Available Methods

  • GET /words → Fetch all words
  • GET /words/2 → Fetch the word where id = 2
  • GET /words/random → Fetch a random word
  • GET /words/random/5 → Fetch 5 random words

How to Use This API:

Python Example

import requests
import json

url = "https://fourapi.onrender.com/quotes/random"

response = requests.get(url)
if response.status_code == 200:
    data = response.json()
    print(json.dumps(data, indent=2))
else:
    print("Error:", response.status_code)

JavaScript Example (Node.js / Browser)

fetch("https://fourapi.onrender.com/words/random")
  .then(response => response.json())
  .then(data => {
    console.log("Fetched word:");
    console.log(JSON.stringify(data, null, 2));
  })
  .catch(error => console.error("Error:", error));

Java Example (with Maven + Unirest)

Add this dependency in your pom.xml:

<dependency>
  <groupId>com.konghq</groupId>
  <artifactId>unirest-java</artifactId>
  <version>3.14.5</version>
</dependency>

Example Java code:

import kong.unirest.HttpResponse;
import kong.unirest.JsonNode;
import kong.unirest.Unirest;

public class FourApiExample {
    public static void main(String[] args) {
        HttpResponse<JsonNode> response = Unirest.get("https://fourapi.onrender.com/colors/random")
                .asJson();

        System.out.println("Response:");
        System.out.println(response.getBody().toPrettyString());
    }
}
