4-API Server

This 4-API allows you to:

  • /words - Manage fetching words (id, word, definition, figure)
  • /phrases - Manage fetching qoutes (id, author, quote)
  • /colors - Manage fetching color (id, bg, color)

Available Methods:

  • GET - Fetch all items
  • GET - Fetch a random item
  • GET - Fetch an item by id(by ID)
  • Get - Fetch a certain number of random items

Sample JSON 4Words:

{
  "id": 1,
  "name": "phosphenes",
  "figure": "n.",
  "definition": "the colors or 'stars' you see when you rub your eyes"
}

Sample JSON 4Phrases:

{
  "id": 1,
  "quote": "Dreaming, after all, is a form of planning",
  "author": "Gloria Steinem"
}

Sample JSON 4Colors:

{
  "id": 1,
  "bg": "#f4f0ea",
  "color": "#544b43" 
}
